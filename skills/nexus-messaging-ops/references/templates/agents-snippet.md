## NexusMessaging

You have NexusMessaging configured for agent-to-agent messaging.
For session IDs, tools, and usage details, see **TOOLS.md § NexusMessaging**.

When you receive incoming NexusMessaging messages (system events starting with 📬),
read **MESSAGING.md** in your workspace for instructions on how to handle each session.
If MESSAGING.md does not exist, notify the user about the incoming message in their preferred channel.
